Вы можете использовать Default для создания новых шаблонов MaxSite CMS:
	- сделайте копию default-каталога для своего шаблона;
	- удалите неиспользуемые компоненты;
	- если LESS не используется, то удалите каталог css-less;
	- свои стили располагайте в каталоге css.
