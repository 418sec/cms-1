* Для установки шаблона, скопируйте его в каталог application/maxsite/templates/ 
после чего выберите его в Основных настройках.
* Всегда используйте последнюю версию MaxSite CMS с гитхаба.
* Для нормальной работы шаблона нужно хотя бы десяток записей на сайте.
* Включите плагин parser_simple.
* В Настройках шаблона выберите
	CSS профили:
		+ fontawesome5-lazy.css
	Дизайн:
		+ opensans.css
		+ robotoslab.css
