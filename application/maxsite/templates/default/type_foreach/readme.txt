В этом каталоге располагаются type_foreach-файлы.

Подкаталог info-top содержит файлы, которые выбираются и подключаются через админ-панель как шаблоны заголовков записей. В этих файлах происходит подключение из my-template, в которых и происходит непосредственный вывод заголовка. Такое разделенеия файлов используется для того, чтобы в info-top-файле можно было комбинировать разный вывод.  

Файлы в info-top предназначены для одиночных записей, кроме файлов начинающихся с «full-» — это заголовки для full-записей (главная, рубрки, метки и т.п.).

Прочие type_foreach-файлы.

info-top-full.php
	Файл шаблона заголовка для full-записей по умолчанию. Используется если ничего не выбрано в настройках админ-панели. 

info-top-page.php
	Аналогично, то для одиночных записей.

page-content-full.php
	Служебные файл, в котором откоючается вывод контента для full-записей по условию.

page-other-pages-out.php
	Файл, который формирует вывод блока «Еще записи по теме». В данном варианте в виде миниатюр.

-mso-info-bottom-page.php
	Файл для вывода навигации next/prev в конце записи. Если она нужна, то переименуйте файл в mso-info-bottom-page.php
