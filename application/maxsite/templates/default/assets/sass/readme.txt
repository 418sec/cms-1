Здесь находятся scss-файлы шаблона.

berry — Berry CSS
bootstrap4 - Bootstrap CSS 4 версии
fontawesome5 - иконки FA 5 версии
fonts - подключение шрифтов
lazy - файлы для css/lazy - отложенная загрузка в конце BODY
menu - менюшки (выбор для админ-панели)
profiles - css-профили (выбор в админ-панели)
template - основные стили шаблона
typography - базовая типографика после Bootstrap
visual - разные визуальные эффекты

style.scss - стили для секции HEAD

Описание см. Организация CSS-кода - https://maxsite.org/page/css-workflow
